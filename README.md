## ⑤工夫した点・こだわった点

・リロードボタンの実装
・勝ちor負けのときにフェードインで文字を出力
・デフォルトの手の画像の明度を下げることでより選択した感をだしたり、
　あいこならカーソルを移動する元にもどる、決着がついたらカーソルを移動しても戻らないようにするなどこまかい部分にこだわった。

## ⑥難しかった点・次回トライしたいこと(又は機能)

面白そうなゲームの見た目にするのと、シンプルなじゃんけんだけではなく追加機能を加えたい。
